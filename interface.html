Knowing the user's interface

Menu Bar with buttons to add new records, edit existing records, or delete records.

  • Add: add new records into the current table. Click on Save button to save the new record.
  • Modify: modify a selected row from the current table. To save changes click again on "Modify" button.
  • Delete: delete a selected row from current table.
  • Log Out: this button logs the user out of the application.

Navigation Pane to switch between the diffent tables or querys.

The main Display Area where shows the complete tables or querys in a structured and organized manner. Shows also, the name of the table that you are consulting.
